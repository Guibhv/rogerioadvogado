Sistema de Gestão para Escritórios de Advocacia

Uma solução completa para gestão de processos, clientes e documentos

Sobre o Projeto

Este é um sistema completo de gestão desenvolvido para escritórios de advocacia, oferecendo ferramentas poderosas para gerenciamento de processos, clientes, documentos e muito mais.

Site Institucional

Páginas profissionais para apresentação do escritório, equipe e áreas de atuação.

Área do Cliente

Acesso seguro para clientes acompanharem seus processos e documentos.

Painel do Advogado

Ferramentas completas para gestão de processos e clientes.

Tecnologias Utilizadas

  • Backend: PHP 8.x
  • Banco de Dados: MySQL
  • Frontend: HTML5, CSS3, JavaScript
  • Segurança: CSRF, XSS, SQL Injection protection

Como Iniciar

  1. Clone o repositório
  2. Importe o banco de dados do arquivo database/schema.sql
  3. Configure as credenciais do banco em config/database.php
  4. Acesse o sistema através do servidor web
